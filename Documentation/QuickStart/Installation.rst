==================
Quick installation
==================

..  attention::

    This extension can only be installed in a composer-based TYPO3 installation!

In a :ref:`composer-based TYPO3 installation <t3start:install>` you can install this extension with following command:

..  code-block:: bash

    composer require r3h6/oauth2-server
